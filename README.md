# VK Mini Apps UI [![NPM][npm-badge]][npm-link]

A package with VK UI components filled up with theming and JSS (no CSS
included).

## Installation
```bash
yarn add vkma-ui
```
or
```bash
npm i vkma-ui
```

## Components
- Button
- ConfigProvider
- DeviceProvider
- FormControl
- Input
- Link
- Select
- Separator
- ThemeProvider
- GlobalStyleSheet
